# Shared Component

Shared types, utilities, and constants used across the monorepo.

## Overview

This component contains:
- TypeScript type definitions
- Shared utilities and helpers
- Common constants
- Validation schemas

## Directory Structure

```
shared/
├── types/           # Shared TypeScript types
├── utils/           # Utility functions
└── constants/       # Shared constants
```

## Usage

### In Python (API/Model)

```python
# Type definitions can be shared via JSON schema
# or pydantic models
```

### In TypeScript (UI)

```typescript
// Import shared types
import { PredictionResponse } from '../shared/types';
```
