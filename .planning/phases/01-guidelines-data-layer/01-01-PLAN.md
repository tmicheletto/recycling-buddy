Produce the label_to_rny.json mapping table that translates each of the 67 EfficientNet classifier labels into the corresponding recyclingnearyou.com.au item-page URL slug (e.g., `glass-bottles-jars` → `glass-bottles-jars`, `coffee-cups-pods` → `coffee-cups`).

Purpose: The GuidelinesService (Plan 02) will use this table to construct the correct RNY URL when grounding an LLM query. Without an accurate mapping, the LLM receives irrelevant page content or the fetch fails with a 404.

Output: `data/label_to_rny.json` — a JSON object with all 67 classifier labels as keys.

- Base: `https://recyclingnearyou.com.au/material/home/{slug}`

- Slugs are lowercase-hyphenated; closely mirror classifier label format in many cases
